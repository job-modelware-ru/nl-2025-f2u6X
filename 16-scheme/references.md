# references.md

Пополняемый список материалов по Scheme и близким диалектам (Guile/Racket), которые используются в презентациях `01–06`.

## Основные стандарты и спецификации

- R5RS: *Revised^5 Report on the Algorithmic Language Scheme* (1998).
- R7RS (small): *Revised^7 Report on the Algorithmic Language Scheme* (2013).
- SRFI (Scheme Requests for Implementation): каталог расширений языка.

## Реализации (для запуска примеров)

- GNU Guile (Scheme): документация и справочник.
- Racket: документация, R5RS/R7RS языковые уровни, онлайн-песочницы.

## Книги

- Abelson, Sussman, Sussman — *Structure and Interpretation of Computer Programs (SICP)*.
- Dybvig — *The Scheme Programming Language*.
- Dorai Sitaram — *Teach Yourself Scheme in Fixnum Days*.

## Онлайн-ресурсы / песочницы

- Онлайн-песочница Racket (подходит для запуска Scheme-стиля кода).
- Replit / подобные онлайн IDE с поддержкой Scheme/Racket/Guile (наличие зависит от платформы).

## По темам (модули, макросы, потоки)

- SRFI-1 (списки), SRFI-9 (records), SRFI-18 (threads) — полезные расширения (если поддержаны реализацией).
